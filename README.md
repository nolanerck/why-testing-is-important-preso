# why-testing-is-important-preso
Sample files from my preso "Why Testing Is Important And Where Do I Start?"

Each of the folders that include demos are meant to be their own Lucee server instance.  

To run the demos you need to have CommandBox installed (or know how to map your Lucee server to point to these folders).

Then do (for example)

cd ./1_MakeShiftTestSuite
box server start
and open the tests.cfm file in your browser
